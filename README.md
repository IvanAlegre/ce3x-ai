# CE3X-AI

**Certificación Energética Inteligente** - Simulador de certificación energética inspirado en CE3X (España).

## 🏠 Características

- Cálculo de certificación energética (letras A-G)
- Estimación de consumo en kWh/m²·año y emisiones CO₂
- Simulación de mejoras con impacto en letra y ahorro
- Intervalos oficiales según normativa española (RD 390/2021)
- Interfaz móvil-first con diseño startup sostenible

## 🚀 Demo

Abre `index.html` en cualquier navegador.

## 📊 Escala energética oficial

| Letra | Consumo (kWh/m²/año) |
|-------|---------------------|
| A | < 44.6 |
| B | 44.6 - 72.3 |
| C | 72.3 - 112.1 |
| D | 112.1 - 172.3 |
| E | 172.3 - 303.7 |
| F | 303.7 - 382.6 |
| G | ≥ 382.6 |

## ⚙️ Stack

- HTML5 + Vanilla JS
- Tailwind CSS (CDN)
- Sin backend - todo cálculo en cliente
